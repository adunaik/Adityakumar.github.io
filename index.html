My Projects

3D content creation and refinement

This project focuses on developing a learning-based algorithm for transforming 2D images into high-quality 3D models, facilitating efficient and realistic 3D content creation for interactive applications in video games, augmented reality (AR), and virtual reality (VR).The gaming industry extensively uses point clouds to build detailed 3D models, creating realistic environments in AR and VR games.Additionally, heritage reconstruction benefits from these techniques by enabling the digital preservation and restoration of historical sites and artifacts.

Human based activity recognition using wifi based CSI signal

This project focuses on human activity recognition using WiFi-based Channel State Information (CSI) signals. It involves collecting CSI data from 2 ESP-32, preprocessing the signals to extract relevant features like amplitude, phase, and frequency shifts, and then training machine learning models such as deep neural networks or support vector machines for activity classification. The project aims to achieve high accuracy in recognizing human activities such as walking, running, sitting, and standing based solely on the variations in WiFi signals caused by these activities

Class specific image to 3D reconstruction

Developed a learning-based algorithm for class-specific image to 3D reconstruction, focusing on generating diverse 3D models from 2D images by varying the parameters of the latent vector. This approach enables the creation of distinct and varied 3D structures within a specific object class.This project contributes to the field of 3D computer vision by providing a method to generate varied and detailed 3D models from simple 2D images, with potential applications in virtual reality, augmented reality, and robotics.

Comparative Analysis of Canny Edge Detection Algorithm Using OpenMP, CUDA, and OpenCL

Parallel Computing Techniques: Conducted a comparative analysis of the Canny edge detection algorithm using OpenMP, CUDA, and OpenCL, evaluating their performance in terms of execution time, efficiency, and resource utilization.
Performance Optimization: Implemented the Canny edge detection algorithm on multi-core CPUs and NVIDIA GPUs, achieving significant speedups, with CUDA achieving the highest performance, followed by OpenCL and OpenMP.
Cross-Platform Compatibility: Demonstrated the trade-offs between performance and portability, highlighting CUDA's efficiency for GPU processing and OpenCL's flexibility for heterogeneous computing environments.
